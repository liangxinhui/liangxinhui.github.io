脚本解析

ls -t | grep "$1" | head -n 1 分一下三个步骤：

  • ls -t:
    按时间排序，最新的文件在第一个。
  • grep "$1" :
    过滤输入的字符串，例如本例中为log_.如果不输入的话，默认为空，表示所有文件。
  • head -n 1:
    返回所有符合规则的文件中的第一条（即最新）。
